# FINAL YEAR PROJECT
## GPT Models for Time-series Analysis and Forecasting.

This repository contains the implementation of my final-year university project, which explores the use of foundation models for time-series forecasting.

The project compares the forecasting performance of:

- Gradient Boosting Model (GBM)
- TimesFM (Zero-shot)
- Chronos (Zero-shot)
- Chronos Fine-Tuned on LoRA Adapters

Experiments are conducted on real-world dataset. Traffic monitoring data was collected from a road intersection in Iași, Romania where six HD IP cameras stream video at 25 FPS
over a private 5G network to a GPU powered Multi-access Edge Computing node running a video analytics pipeline.

Model performance is evaluated using common forecasting metrics such as MAE, MAPE, and WAPE.
